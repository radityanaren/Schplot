# Schplot

A simple Python data plotter that based on PyQt5 and Matplotlib, which aims to simplify minor works that requires Python visualisations. 

## Features
- Load CSV or Excel data
- Plot multiple data series
- Add regression lines (Linear, Exponential, Logarithmic)
- View regression statistics
- Save simple plot code

## Installation
**Make sure you have Git and Python 3.7 or better installed on your computer**
1. Clone the repo :
   ```bash
      git clone https://github.com/radityanaren/Schplot.git
   ```
2. Go to directory :
   ```bash
      cd schplot
   ```

2. Install dependencies :
   ```bash
      pip install .
   ```

3. Run the app :
   ```bash
      python schplot.py
   ```
   or as a module :
   ```
      python -m schplot
   ```
   
Or you can double click schplot.py
